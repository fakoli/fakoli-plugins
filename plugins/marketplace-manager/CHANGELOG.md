# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-02-01

### Added
- `add-plugin` command for creating new plugins from template
- `remove-plugin` command for removing plugins from the marketplace
- `install-workflows` command for installing GitHub Actions workflows
- install_workflows.sh script for workflow installation

## [1.0.0] - 2025-02-01

### Added
- Initial release
- marketplace-manager skill for managing the fakoli-plugins marketplace
- add_plugin.sh script for creating plugins from template
- remove_plugin.sh script for removing plugins
- marketplace_status.sh script for checking marketplace status
- Complete workflow documentation for plugin management
